# 📍 Web Geolocation Tracker

Web untuk melacak dan menampilkan lokasi pengguna secara real-time di atas peta digital menggunakan **Leaflet.js** dan **OpenStreetMap**.

## 🚀 Fitur Utama
- **High Accuracy:** Memaksimalkan sensor GPS pada perangkat untuk akurasi titik.
- **Accuracy Radius:** Menampilkan lingkaran biru sebagai indikator area akurasi GPS.
- **Loading State:** Memberikan instruksi kepada pengguna saat sistem sedang mencari sinyal.
- **Responsive Design:** Tampilan penuh (Full Screen) di HP maupun Desktop.

## 🛠️ Teknologi
- **Leaflet.js** (Library Map yang ringan)
- **HTML5 Geolocation API** (Native browser)
- **OpenStreetMap** (Data peta gratis)

## 📖 Cara Penggunaan
1. Buka link Demo atau jalankan file `index.html`.
2. Saat muncul notifikasi **"Allow location access"**, klik **Allow/Izinkan**.
3. Peta akan otomatis bergerak ke titik koordinat Anda.

## ⚠️ Catatan Privasi
Aplikasi ini hanya berjalan di sisi klien (*Client-side*). Data lokasi Anda tidak disimpan di server mana pun dan hanya digunakan untuk ditampilkan di browser Anda sendiri.
